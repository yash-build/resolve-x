# ResolveX

ResolveX is an AI-powered campus issue resolution platform designed to improve governance and student well-being in colleges.

## Features

- Report campus issues
- Track issue resolution
- AI-based issue classification
- Role-based governance system

## Roles

Student → Report issues  
Executor → Resolve issues  
Committee → Review and escalate  
Admin → Governance and monitoring

## Tech Stack

Frontend: React.js  
Backend: Node.js  
Database: Firebase / MongoDB
